

# 🎮 Tic-Tac-Toe (Desas)
![island](https://github.com/Lvumba1234/Documentation_prog_lev/blob/main/island.jpg)
Interaktīva un moderna "Desu" spēle, kas izstrādāta, izmantojot tīru JavaScript, HTML un CSS. Spēle ir pilnībā funkcionējoša ar uzvaras loģiku, neizšķirta noteikšanu un responsīvu dizainu.

## ✨ Galvenās iespējas
- **Divu spēlētāju režīms:** Spēlētāji (X un O) veic gājienus pēc kārtas.
- **Uzvaras noteikšana:** Automātiska pārbaude pēc katra gājiena, izmantojot iepriekš definētus uzvaras modeļus (*win patterns*).
- **Neizšķirts:** Spēle atpazīst, ja visi 9 lauciņi ir aizpildīti bez uzvarētāja.
- **Dinamisks stils:** Katram spēlētājam ir sava atšķirīga krāsa (O - balta, X - ciāna).
- **Spēles atiestatīšana:** Iespēja sākt jaunu spēli jebkurā brīdī ar "Reset" vai "New Game" pogām.

## 🛠️ Tehnoloģijas
Projekts ir veidots bez ārējām bibliotēkām, izmantojot:
* **HTML5** – spēles struktūrai.
* **CSS3** – vizuālajam noformējumam un Flexbox izkārtojumam.
* **JavaScript (ES6)** – spēles loģikai un notikumu apstrādei.

## 📸 Izskats
Spēle izmanto modernu tumšo tēmu ar šādām krāsām:
- **Fons:** Deep Purple (`#5e548e`)
- **Lauciņi:** Dark Navy (`#231942`)
- **Spēlētājs X:** Cyan
- **Spēlētājs O:** White
